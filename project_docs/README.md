# Punelok E-Delivery - Project Documentation

Welcome to the documentation for the Punelok E-Delivery project.

## Overview
This platform is an e-delivery application with a focus on seamless user experience, order tracking, and efficient delivery management.

## Project Structure
- **/client**: The frontend client application.
- **/server**: The backend NestJS application (TypeScript).
- **/project_docs**: Project documentation and guidelines.

## Tech Stack
- **Backend**: NestJS, Node.js, TypeScript.
- **Frontend**: Defined within the client directory.

## Getting Started

### Backend Setup
1. Navigate to the `server` directory: 
   ```bash
   cd server
   ```
2. Install dependencies: 
   ```bash
   npm install
   ```
3. Run the development server: 
   ```bash
   npm run start:dev
   ```
   The backend will start and watch for changes on your local machine.
